# igni_save

Igni Save adalah aplikasi pencatat tabungan berbasis komunitas dan gamifikasi yang dirancang untuk mengubah kebiasaan menabung yang membosankan menjadi sebuah "misi" yang seru. Dengan sistem streak, peringkat, dan verifikasi fisik, Igni Save memastikan setiap rupiah yang Anda sisihkan tercatat dengan penuh kebanggaan.
